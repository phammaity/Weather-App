# Weather-App

Xcode 10.1 or newer

iOS 12.1 or newer

npm  6.4.1 and node 10.15.3


2 ways to run app:

By Xcode :

Step 1: run “$npm start” in the root directory.

Step 2: press Run button on Xcode



By terminal:

Step 1: run “$npm start” in the root directory.

Step 2: run “react-native run-ios”


API :https://www.metaweather.com/api/
